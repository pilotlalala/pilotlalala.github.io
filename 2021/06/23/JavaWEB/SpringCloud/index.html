springcloud

定义/作用

springcloud管理springboot的分布式系统

完美支持springboot

而且免费

核心组件

Eureka Server

Eureka client

Discovery Client

Ribbon 自动负载均衡

Feign

Breaker 熔断

Zuul

详见：https://blog.csdn.net/xunjiushi9717/article/details/91988479

总归是负载均衡 熔断

微服务集群

复用的功能抽成单独的服务来管理 提高可用性 可扩展性，提高容错率

这样的服务就是所谓的微服务

缺点：

架构复杂

运维费用高

搭建

选择springboot init

然后选择discovery里面的erukaserver 搭建注册中心

选择erukaclient搭建客户端

记得在main方法上加上对应的注解

先开注册中心再开客户端

在java包里面建controller

@RestController或者@Controller

client之间互相访问

@RequestMapping("/say")
public void say(){
    RestTemplate restTemplate = new RestTemplate();
    String resultCall = restTemplate.getForObject("http://localhost:8763/hello/say",String.class);
    System.out.println("返回结果："+ resultCall);
    System.out.println("say");
}
