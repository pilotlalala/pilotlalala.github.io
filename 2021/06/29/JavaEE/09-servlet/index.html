09-servlet

发表于 2021-06-29
学习 > Java > JavaEE-学习java
派洛特

Servlet

补充

记得把jsp文件的编码改成UTF-8

改jsp的路径

下面overview右边的module里面的edit把path改成自定义的就可以了

Servlet

servlet applet 
运行在服务器端的小程序
就是一个接口定义了java类被浏览器访问到的时候的规则

入门

写一个类实现接口 导入servlet.servlet接口
然后servlet有一些抽象方法

然后再web.xml文件中 <servlet>块里面写
<servlet-name>名字</servlet-name>
<servlet-class>
		全路径 比如说cn.gockteck.文件名
</servlet-class>

然后在<servlet-mapping>
		<url-pattren>
			/servlet1访问的时候写什么
		</url-pattren>
</servlet-mapping>

生命周期

init创建只执行一次

然后执行service方法  之后都直接service
用来执行请求

停止前执行destory方法
设置servlet初始化时间点
<load-on-startup>
	时间  默认为负值 负值是调用的时候才会执行init 
	0或者正整数就是随着tomcat启动就初始化
</load-on-starup>
把一串话放到tomcat-users.xml里面
然后点进入的manage
输入账号密码
可以控制启动和停止
<init-param>参数信息
	<param-name>name</>
	<param-value>value</>
</>

servlet体系结构

servlet
genericServlet 对servlet的四个方法做了实现
				没有对service做实现 
HttpServlet
主要用这个
几种传过来的方法对应do方法
比如说doGet()和doPost()
实现了这个接口然后重写对应的do方法就可以了
主要是doGet和doPost

servlet3.0以上支持注释 
不用web.xml了
注释:
@webServlet("url-pattern怎么写就怎么写")

@webServlet(urlPattrens={"/servlet名","名字2"})

配置loadon-starup:
@webServlet()

get请求和post请求

get:中文不会乱码
post:中文会乱码

原理：HttpServlet里面的service方法里面写了equals判断是哪一种请求
获得前端的值
获取值
String name = request.getParameter("name");

请求设置编码
request和respnse都会被封装在参数里

req.setCharacterEncoding("UTF-8")
响应设置编码
resp.setContentType("text/heml;charset=UTF-8")

resp.getWriter().print()可以输出值
