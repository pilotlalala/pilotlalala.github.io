前言

产品分类

系统软件类：Linux、苹果系统、maxos
游戏类：
应用类型：**系统、淘宝京东

C/S和B/S

B：browser S：server C：Client
B/S：网络+浏览器
C/S：客户端（QQ、迅雷等）

HTML

网页三剑客

结构：html技术实现
表现：网页外观 css样式决定
行为：网页功能，javaScript实现

html基础

超文本标记语言（Hyper Markrup Language）

标签

<p>段落标签，前后会有空格</p>
<h1~6>标题标签</h1~6>
<br/>换行标签  <hr/>分割线
<pre>格式预定义标签，编码器如何显示就如何显示</pre>
<strong>加粗</strong>   <em>斜体</em>
<span>效果和直接写在编码器一样，但是可以加标签什么的</span>

标签分类

自闭合：没结束
普通：有结束

图片

<img src="路径" widh="宽度 可以固定像素px或者百分比" height="高度同上" alt="未加载时显示文本"/>  
路径
 绝对路径
 相对路径

字体

<em>
<strong>

列表标签

<ol>
 <li>有序列表</li>
</ol>
<ul>
 <li>无序列表</li>
</ul>
<dl>
 <dt>
   <dd>定义列表</dd>
 </dt>
</dl>

表格

<table  border="边框、像素"  bgcolor  cellspacing="单元格之间的空格" cellpacing="单元格和里面内                                                                               容之间的格">
 <thead>表头
  <th>表头标题，字体加粗居中</th>
 </thead>
 <tr rowspan="行合并">行
   <td  colspan="合并列"  align=”字的位置“>列</td>
 </tr>
 <tfoot>表脚</tfoot>
</table>

表单

属性

method：提交方式 post、get
action：表单跳转的路径
enctype：文件上传、音频视频需要设置的属性

input标签

属性

type=

text 文本框  value="默认值" placeholder="提示" 
password 密码框
checkbox 复选框 多个 
radio 单选框 name="名字 多个名字相同时会互斥"
file 文件上传
email 邮箱
button 按钮 
reset 重置
submit 提交 用的最多！！！！
hidden 隐藏标签 把input隐藏起来

value=””

表示预先设定的相关之

id=

用来标识id，通常不能重复

class=

用来标识class名

超链接

<a href="跳转的页面" 
   taget="跳转的形式 _self当前 _top跳到顶层框架 _parent跳到当前框架的父框架 _search重新打开一个页面，如果有了它不会打开新的  _blank重新打开一个新的空白窗口进行跳转" 还可以是具体某个框架名></a>

锚链接

href="#id名字"

移动

marquee
