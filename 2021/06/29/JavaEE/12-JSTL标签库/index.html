12-JSTL标签库

发表于 2021-06-29
学习 > Java > JavaEE-学习java
派洛特

JSTL标签库

定义

JSP标准标签库 
jsp的标签机和 封装了jsp应用的通用核心功能
场合EL表达式结合起来使用

EL表达式

定义

expression language
用于查找作用域中的数据 （取值）
然后对他们执行简单操作 可以写在jsp文件的任何位置

组成

标识符 存取器 文字 运算符

11个标识符

11个隐式对象

存取器

el表达式存取器

[] 数组
. 主要用来获取对象属性  比如创建一个java对象 然后存进session.setAttribute  就可以获取对象的属性

运算符

empty
 是否为空 是返回true 否则false
 ${empty 对象}
 使用场景：导航栏中判断用户是否注册

还有其他常见的运算符像==

使用

通过EL表达式取request作用域的值

先request.setAttribute("",)

${name} =<%pageContext.findAttribute%> = <%request.getParameter("name")%>
${array[0]}
如果要指明那个作用域 就在前面加上 作用域. 就可以

JSTL

类别

核心标签 C标签
格式化标签 用来格式化数据  格式化 日期、数字什么的
SQL
XML
HSTL

JSTL-Core标签库

要先引入jar包
<% taglib uri="http://java.sun.com/jsp/jstl/core"  prefix="c"%>
操作作用域
<c:out value=${}></c:out>
输出作用域中的值
<c:set scope="指定作用域" value="变量值" var="变量名" ></c:set>
然后输出就可以了
<c:remove var="变量名"></c:remove>
条件控制
单分支
<c:if test="条件判断 比如说${age==18}">
</c:if>
多分枝
<c:choose>
	<c:when test="条件">
		相当于if
	</c:when>
	<c:when test="">
		相当于else if
	</c:when>
	<c:otherwise>
		相当于else
	</c:otherwise>
<c:choose>
循环控制
遍历 循环迭代 只要便利数组或者集合
使用场景 经常在商品首页循环遍历

<c:forEach begin="开始的值" end="结束的值" step="步长" var="变量的名字" varStatus="下标的名字">
	${变量的名字}获得遍历道德之
	${下标的名字.index}获取下标
	${下标的名字.count }计数
</c:forEach>

遍历数组
<c:forEach items="${数组名} 存在作用域里的" var=取去一个对象的变量名" >
	${去一个对象的变量名.属性}
</c:forEach>
以表格形式遍历集合
<c:forEach items="${数组名} 存在作用域里的" var=取去一个对象的变量名" varStatus="名字">
	<tr>
		<td>
			${vs.count}
		</td>
		<td>
			${去一个对象的变量名.属性}
		</td>
		<td>
			<a href="?" > 修改删除<>
		</td>
	</tr>
</c:forEach>
格式标签库

时间格式

<fmt:formatDate value="${Date对象的名字}" pattren="yyyy-mm-dd 格式"/>

数字格式

<fmt:formatNumber value=“”pattern=“###.00”/>
