EE综合案例

数据库

创建表的实体类在entity文件夹里面
utils包装JDBC工具类
dao包里面写增删改查抽象方法的接口
	增加可以直接传一个实体类作为参数 这样就可以直接获取值
	删除和修改可以直接传id 返回int
	查询分为精确查找和查询所有和模糊查询
	精确查找返回实体
	查询所有返回集合
	模糊查询根据具体的关键字查找，传一个实体类作为参数，然后用集合接收返回值
	查询总记录数 用来分页
在dao包里面建立impl包 写各种具体的操作方法
service包里面建立impl 东西和dao差不多 主要是为了更加复杂的业务处理 然后调用的时候直接调用service这一层
记得servlet里面增删改查的方法中重复的代码（比如说获得数值）可以封装起来 这样代码不会太冗余

	先<c:if test="${}">判断request中的表格 如果为空就用<c:redirect uri="">来重定向到servlet
	使用表格
	先把大表格写好
	然后行添加用c标签的foreach来遍历servlet中查询出来的list
	
	注意sql语句不能写错 不然列表一直是null的话会报错

删除记得重定向

jdbcutils详解

分页

总记录数 必须是满足条件的筛选  select count(*) from 表
每页的条数  
	sql语句：limit 1,10 
				每页显示10条数据  
				第一个参数：第几行数据，和index很像，从0开始
				第二个参数：要查询的数据条数
总页数 总记录数/每页条数
开始行的计算公式
(当前页数-1)*每页条数

分页工具类PageDao

使用

设置总记录数
pageDao.setRscount(int类型的参数) 设置总记录数

获取开始行
pageDao.getPageSize()每页条数
pageDao.getCurrentSize()获取当前页数

在模糊查询的sql基础上加上
sql+= "limit" + 开始行 + "," +每页记录数 分页工具上面的pageSize 

获取分页条样式
String pageTools = pageDAO.pagetool(PageDao.BbsImage)或者text或者BbsText 
把样式添加到request里面
