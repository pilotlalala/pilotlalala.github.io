加入购物车的实现方式

1.存到session中，购物车中的页面直接从session中取值

2.使用cookie cookie+数据库 客户端

如何实现

一个用户一个购物车 购物车有外键userid

购物车里面有多个商品  一种商品也可以购买多个 把购物车当成集合  每一个元素就是表示商品和对应的数量 List<car>表示购物车

类
1.商品类 名称 价格 id 商品图片 商品简介 
2.购物车里面的每一件商品的对象  商品类的实体类 数量
	重写equals方法 object instanceof Car然后判断得到的id是否相等 如果相等就表示商品相等
3.购物车类 总数量  总价格 商品信息和数量的集合

流程

商品详细页传给接口商品的id和数量
接口给session中的购物车添加商品
购物车中写添加商品方法 通过id查找商品 然后判断商品是否已经存在之后存入购物车
	商品是否存在 shopCar.indexOf(bean){
						int index = shopCar.indexof(bean)然后get出来商品之后数量增加
									}
									
修改商品数量
按下加减按钮然后修改框里的数值 然后ajax跳转 动作属性传值 就可以做到不刷新也可以修改商品信息
