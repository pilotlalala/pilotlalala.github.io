js操作dom对象

DOM对象

DOM：document  object Model 文档对象模型 html和xml的应用编程接口
dom：把网页和脚本以及其他编程语言联系起来
html-dom：html文档对象模型，定义了访问和处理html文档的标准方法
html：

节点

文档节点：文档
元素节点：比如a标签
属性节点：比如href属性，id属性，name属性等。             
    getAttributeNode("")     获取属性节点，比如可以获取某个标签再
    加上.getAttributeNode("value")获取它的value属性节点
    
注释节点：html文档中的注释
文本节点：比如标签里面的文字

                   childNodes    获取文本节点，返回值是数组

                   比如：var span=document.getElement("sp").childNodes;

                   span[0].nodeValue就是这个span里的文本内容

节点的三个属性、方法

 nodeName：节点的属性名称，返回字符串
  nodeType：节点的类型，返回整数，1元素 2属性 3文本
nodeValue：节点的值。元素节点的节点若没有value值则返回null。
                                   文本节点的nodeValue就是文本的内容

js中的dom方法

appendChild（子元素）
removeChild（子元素）
setAttribute（key，value）
getAtttribUte()
createElement()创建指定元素
childNode 返回数组

正则表达式

java中匹配正则表达式的是match方法
js中用的是text方法
不能以双引号开头

普通方式

var reg = /表达式/附加参数
\d 代表0-9中的一个数字 等同于【0-9】
\D 代表飞0-9
\w 表示所有单词 数字和字母都算
\W 表示非单词
^.. 以.....开头
..$ 以.....结尾
\. 代表"."
{至少多少字符，至多多少字符}
ab{2} 一个a跟这两个b
ab{2,}表示有一个a至少跟着两个b
ab{3,5}a跟着3-5个b
* 匹配前一个字符0次或者多次
+ 匹配前一个字符1次或者多次
？匹配0个或者一个字符 

比如 /^[0-9]?b$/  就是表示数字有0个或者1个数字然后以b结尾
【^】 表示否定 中括号里面

表达式操作

jQury和核心框架及函数

jQuery的概念

jQuery的来源

js的弊端：

复杂的文档对象模型

浏览器不一致

缺乏便捷的开发
jq简化了js的开发，是js的封装类库
需要引入jq的封装类 jar包

jq的特点

轻量级
强大的的选择器
出色的dom封装
可靠的事件处理机制
完善的ajax

jQuery写法

先引入jar包

	语句，这是页面加载完就调用;
})

$(function(){
	语句，这是页面加载完就调用;
